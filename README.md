# TikTok → Telegram Бот (облачная версия для Railway)

Бот работает в облаке 24/7, не нагружает ваш компьютер.

---

## Пошаговая инструкция

### Шаг 1 — Создайте Telegram-бота

1. Откройте Telegram
2. Найдите @BotFather
3. Напишите /newbot
4. Придумайте имя и username
5. Скопируйте токен (строка вроде 123456789:AABBccDD...)

### Шаг 2 — Зарегистрируйтесь на GitHub

1. Зайдите на https://github.com
2. Создайте аккаунт (если нет)
3. Нажмите "+" → "New repository"
4. Название: tiktok-bot
5. Оставьте Public, нажмите "Create repository"

### Шаг 3 — Загрузите файлы на GitHub

На странице нового репозитория:
1. Нажмите "uploading an existing file"
2. Перетащите ВСЕ файлы из папки проекта:
   - bot.py
   - tiktok_scraper.py
   - requirements.txt
   - Procfile
   - runtime.txt
3. Нажмите "Commit changes"

### Шаг 4 — Зарегистрируйтесь на Railway

1. Зайдите на https://railway.app
2. Нажмите "Login" → войдите через GitHub
3. Нажмите "New Project"
4. Выберите "Deploy from GitHub repo"
5. Выберите ваш репозиторий tiktok-bot
6. Railway начнёт деплой

### Шаг 5 — Добавьте токен бота

1. В Railway откройте ваш проект
2. Перейдите во вкладку "Variables"
3. Нажмите "New Variable"
4. Name: BOT_TOKEN
5. Value: вставьте токен от BotFather
6. Нажмите "Add"
7. Railway автоматически перезапустит бота

### Шаг 6 — Проверьте

1. Откройте вашего бота в Telegram
2. Напишите /start
3. Если бот ответил — всё работает!

## Команды бота

| Команда    | Что делает                                    |
|------------|-----------------------------------------------|
| /start     | Приветствие                                   |
| /add       | Добавить хэштег (проверит существование)      |
| /remove    | Удалить хэштег                                |
| /list      | Показать все отслеживаемые хэштеги            |
| /search    | Найти видео прямо сейчас                      |
| /interval  | Изменить интервал (например /interval 30)     |
| /help      | Помощь                                        |
